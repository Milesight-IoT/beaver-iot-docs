# 简介

Beaver IoT 前端是 Milesight 开发者服务部开源的前端产品，服务于 Beaver IoT 应用。当前项目中有一个 Web 应用及数个子依赖库。包含了基础业务功能，包括登录、注册、集成、设备，以及仪表盘等，可支撑基础的 IoT 应用快速集成。

- 文档地址：[Beaver IoT Docs](https://docs.beaver-iot.com/)
